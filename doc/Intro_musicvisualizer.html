library(musicvisualizer)

musicvisualizer

The musicvisualizer package retrieves and filters data that contains an album’s features from Spotify. It displays all these features in one plot.

Visualize Album’s Features

Given an artist’s name and ONE of their album’s name, the musicvisualizer() function retrieves a raw data that is later filtered down to a few features (danceability, valence, …) for the album.

library(musicvisualizer)
##basic example code

first_musicvisualizer <- musicvisualizer("Olivia Rodrigo", "GUTS")
first_musicvisualizer
